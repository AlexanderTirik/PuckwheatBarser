# PuckwheatParser

Frontend prod: https://puckwheat-barser.vercel.app/
Backend prod: https://puckwheatbarser-backend.herokuapp.com

## backend:
1.  ```cd backend``` 
2. ```pip3 install -r requirements.txt```
3. ```cd src```
4. ```python3 app.py```

or using docker:
1. ``` docker-compose build ```
2. ``` docker-compose up ```

## frontend:
1. ```cd frontend```
2. ```npm install```
3. Change file ```.env.example``` to ```.env```
3. ```npm start```
